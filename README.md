# GTA-Helper

AHK Script for GTA Online

Script include usage of [ViGEm](https://github.com/ViGEm/ViGEmBus/releases) for controlling game in background by emulating virtual Xbox360 / DualShock 4 controllers

## Setup
1. Install [ViGEm](https://github.com/ViGEm/ViGEmBus/releases)
1. Download from the script and Lib
1. Run the script
